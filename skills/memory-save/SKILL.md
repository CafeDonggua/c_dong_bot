# Skill: 記憶保存

## Summary
- 讓使用者把指定內容保存到當日記憶檔案

## Triggers
- 使用者想要「記住」某件事
- 使用者明確要求保存一段文字

## Inputs
- 記憶內容（文字）
- 可選：指定日期（YYYY-MM-DD）

## Outputs
- 已寫入的記憶檔案（當日或指定日期）

## Steps
1. 接收使用者輸入內容
2. 判斷是否包含指定日期
3. 將內容寫入對應日期的記憶檔案
4. 回覆保存成功訊息

## Constraints
- 僅處理文字內容
- 不支援多媒體附件

## Safety
- 避免保存敏感個資（如身分證、信用卡）
- 若內容疑似高度敏感，提醒使用者改用安全管道

## Examples
- 正例：請記住今天要買牛奶
- 反例：請記住我的信用卡號碼 4242-4242-4242-4242
