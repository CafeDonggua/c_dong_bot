# Skill: 記憶查詢

## Summary
- 讓使用者查詢既有記憶，支援當日或日期區間

## Triggers
- 使用者想要「回想」記憶
- 使用者提供日期或日期區間

## Inputs
- 查詢關鍵字（可選）
- 日期（YYYY-MM-DD）或日期區間

## Outputs
- 符合條件的記憶條目清單

## Steps
1. 解析使用者輸入中的日期或區間
2. 讀取對應日期的記憶檔案
3. 依關鍵字篩選結果（若有）
4. 回覆整理後的記憶內容

## Constraints
- 僅能查詢已存在的記憶檔案
- 若日期無資料，回覆無結果

## Safety
- 避免揭露他人資料或未經同意的內容
- 若查詢範圍過大，提示縮小範圍

## Examples
- 正例：回想今天的記憶
- 反例：查詢所有人的記憶（超出範圍）
