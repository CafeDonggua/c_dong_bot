# Quickstart: 網路搜尋彙整

## 目的

在 Telegram 個人助理中提供關鍵字搜尋與連結摘要彙整能力。

## 前置條件

- 已設定必要環境變數（`OPENAI_API_KEY`、搜尋 API Key、`TELEGRAM_BOT_TOKEN`）
- 已安裝專案相依套件

## 執行步驟

1. 啟動機器人
2. 輸入「/search 關鍵字」觸發搜尋彙整
3. 輸入「/summary 連結」觸發連結摘要

## 測試

- 執行 `pytest` 驗證回覆格式與錯誤處理
- 手動測試 3 個關鍵字與 2 個連結

## 驗收提示

- 回覆格式包含摘要、重點、來源
- 無結果時有清楚提示與改寫建議
